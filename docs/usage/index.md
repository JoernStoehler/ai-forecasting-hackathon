# Usage

This site documents how to run and use the project locally.

- Run all services: `bash scripts/run.sh --actions start --services frontend backend docs`
- Stop services: `bash scripts/run.sh --actions stop --services frontend backend docs`
- Status: `bash scripts/run.sh`

Default ports:
- Frontend: http://localhost:5173
- Backend:  http://localhost:8080
- Docs:     http://localhost:8000
